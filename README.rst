Providers
=========
Set of microservices implemented with different technologies

Setup
-----
Run

.. code:: bash

  ./bin/install [framework]

where framework can be *flask* (**), *django* (*d*) or *pyramid* (*p*).

Test
----
Run

.. code:: bash

  ./bin/test [framework]

where framework can be *flask* (**), *django* (*d*) or *pyramid* (*p*).
